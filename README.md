# BowlingTech — homepage revizuit

Homepage cinematic, orientat spre investitorul care construiește un centru nou. Povestea urmărește traseul „spațiu → valoare → plan → sistem → experiență → discuție → dovadă → autoritate → suport”.

## Tehnologie

- HTML semantic;
- CSS modern cu scene `sticky`;
- JavaScript fără framework;
- SVG și materiale video/imagine pentru scenele tehnice;
- scroll nativ, fără blocarea derulării;
- fără WebGL și fără dependențe externe obligatorii.

Site-ul poate fi publicat ca proiect static sau într-un director Apache/XAMPP.

## Instalare în XAMPP

1. Păstrează o copie a folderului actual.
2. Extrage conținutul arhivei în `C:\xampp_lite_8_5\www\bowlingtechexpert\`.
3. Nu copia folderul `bowlingtech-new` ca subfolder; `index.html` trebuie să rămână direct în `bowlingtechexpert`.
4. Deschide `https://localhost/bowlingtechexpert/` și reîncarcă pagina fără cache.

## Fișiere principale

- `index.html` — structura și conținutul;
- `styles.css` — designul de bază și scenele existente;
- `styles-v2.css` — revizia UX, responsive și accesibilitate;
- `app.js` — storytelling, configurator și proiecte;
- `confidentialitate.html` și `cookies.html` — pagini juridice de bază;
- `assets/images/` — fotografii și elemente de brand;
- `assets/media/` — materialele generate pentru hero, asamblare, suport și footer.

## Elemente implementate în revizie

- hero întunecat la început, luminat progresiv prin scroll;
- CTA clar către configurator, fără alegerea prematură dintre trei servicii;
- secțiune de valoare regândită ca întrebări ale cumpărătorului;
- plan fără sliderul decorativ pentru numărul de piste;
- scenă de experiență fără animația „X / strike”;
- bandă de autoritate cu afirmațiile publicate de BowlingTech;
- hartă SVG și lista orașelor publicate în portofoliu;
- suport post-instalare explicat ca traseu operațional;
- CTA fix care se ascunde în zona formularului și a footerului;
- buton de revenire la început și bară roșie de progres;
- datele firmei și pagini juridice minime.

## De verificat înainte de publicare

- validarea juridică a paginilor de Confidențialitate și Cookies;
